## Installation

1. Add repository: `https://github.com/ArrestedOps/ha-ea-dashboard`
2. Install EA Trading Dashboard v3.3.0
3. Install MT4 EA: `HA_TradeSync_MT4_v3.3.mq4`
4. Configure EA with your webhook URL
5. Done!
