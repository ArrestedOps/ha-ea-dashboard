# EA Trading Dashboard Add-on

Home Assistant add-on for tracking MetaTrader 4/5 Expert Advisors.

## Installation
1. Add repository to Home Assistant
2. Install "EA Trading Dashboard"
3. Configure webhook secret (optional)
4. Start add-on
